# Physical-Design Common Collateral

This directory is the shared physical-design home for reusable floorplan, IO, timing,
placement, routing, and signoff collateral.

The current repository stage emits deterministic foundation PD review artifacts
from builder actions: DEF stages, final DEF, foundation GDSII/SPEF, informational
reports, and layout images. Real OpenROAD/OpenROAD-flow-scripts signoff execution
is intentionally deferred until technology collateral and backend bootstrap are
introduced.
